Arch 配置双显卡

ArchLinux 双显卡配置，应该多数 Linux 系统都适用，可能细节不一样。

首先开机时禁用 N卡驱动 nouveau

开机启动时如果开机黑屏按 e 加入以下

modprobe.blacklist=nouveau

安装 NVIDIA驱动

sudo pacman -Syu nvidia nvidia-settings xorg-xrandr

配置

修改 grub

sudo nvim /etc/default/grub

GRUB_CMDLINE_LINUX_DEFAULT="loglevel=3 quiet nvidia-drm.modeset=1 acpi_osi=! acpi_osi=\“windows 2009\””

修改完之后重新生成 grub.cfg

sudo grub-mkconfig -o /boot/grub/grub.cfg

添加 initramfs 模块

Arch Wiki initramfs

在 /etc/mkinitcpio.conf 中找到 MODULES=() 加入

nvidia nvidia_modeset nvidia_uvm nvidia_drm

重新生成一下 initramfs

sudo mkinitcpio -P

add pacman hook

每次更新 NVIDIA 都需要重新生成 initramfs 可以使用 pacman hook

sudo nvim /etc/pacman.d/hooks/nvidia.hook

[Trigger]
Operation=Install
Operation=Upgrade
Operation=Remove
Type=Package
Target=nvidia
Target=linux
# Change the linux part above and in the Exec line if a different kernel is used

[Action]
Description=Update Nvidia module in initcpio
Depends=mkinitcpio
When=PostTransaction
NeedsTargets
Exec=/bin/sh -c 'while read -r trg; do case $trg in linux) exit 0; esac; done; /usr/bin/mkinitcpio -P'

显示管理器

Arch Wiki

bumblebee

如果是 linux-lts 是安装 bumblebee bbswitch-dkms

sudo pacman -S bumblebee bbswitch
sudo gpasswd -a user bumblebee
systemctl enable bumblebeed.service
