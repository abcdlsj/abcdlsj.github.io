开始搭建

这里我们要使用到的是v2ray.fun脚本,其实使用脚本对于自己的提升不大,所以可以的话还是自己配置文件或者自己写脚本(说的轻巧w(ﾟДﾟ)w)

登陆vps

搭建之前,我们看一下Readme

发现最好使用Xshell才能导出配置文件

但是我们之前生成的是PuTTY密匙,后缀是.ppk不能直接使用

所以我们需要转换成Xshell的密匙文件

我们打开PuTTYgen,前面的«搭建酸酸乳»有提到过

点击conversions选择import key

选择你之前生成的.ppk文件

之后点击Export OpenSSh key

打开Xshell(xshell下载地址)

点击新建

在下面这栏,填入主机IP地址

点击用户身份验证,然后照下图选择,密匙选择你刚刚生成的,密码不需要填,因为我们导出时没有设置密码

点击连接or确定

然后就连接成功了

使用

完整v2ray客户端

最新版本的获取方式

(不会就用上面的完整客户端就好)：点击Github下载 v2rayN.exe ，同时Github下载 v2ray-windows-32.zip均下载页面最近Releases的版本。将下载回来的v2ray-windows-32.zip解压，然后将下载回来的v2rayN.exe执行文件复制到v2ray-windows-32.zip解压后的文件夹内(有大约10个文件一起的文件夹即进入到最终一个文件夹)，然后双击运行v2rayN.exe，遇拦截请点更多信息 → 仍要运行
